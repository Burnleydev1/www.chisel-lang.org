Related Docs: object Scheduler | package executable

class Scheduler extends LazyLogging

The scheduler holds the assignment statements of the entire circuit. Clocks introduce a level of complexity, in that when they flip registers who use those clocks must copy there inputs to their outputs. Also since registers are triggered by the derived highest level clock that can be found, child clocks must be separately driven. This is what the triggeredUnassigns are for.
